Nexmo Client SDK for iOS

Overview

The Client SDK serves as your gateway for integrating communications in your app.
With a set of features including IP-IP calls, IP-PSTN/PSTN-IP calls, chat messages and more, you can easily address your communications needs.

Nexmo Client SDK for iOS current version is 2.1.8.

Setup

To get started with the Nexmo Client SDK for iOS, check out the Nexmo Mobile Developer Guide for iOS.
